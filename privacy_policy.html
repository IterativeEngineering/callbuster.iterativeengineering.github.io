Last update: 08.2024 This privacy policy applies to the use of the Call Buster - Free Spam Call Blocker app on Android. **Personal data** The Call Buster - Free Spam Call Blocker application does not request any personal data from the user, not does it receive personal data by technical means or store it. Moreover, the application does not receive, use or store any phone identifiers, such as the SIM card numbers, IMEI and MAC addresses, or any other IDs. This data is not required for the application to function. **Android System Permissions** The Call Buster - Free Spam Call Blocker is designed to request and use the minimal number of permissions. The application requires only 1 mandatory permission and 1 recommended permission to be fully operational. For Android 10 and above, the “Default caller ID & spam” permission is required. It allows the Call Buster - Free Spam Call Blocker app to obtain the incoming call number and check it in the database of unwanted numbers. Call Buster - Free Spam Call Blocker will not replace the standard call handling application. There will be no changes to the interface. Call Buster - Free Spam Call Blocker, as well as other applications with similar functionality, will not work without this permission. The "Display over other apps" permission is not required for the application to work. However, this permission will be needed if the user wants to enable the "Show info window" feature. This function allows you to display the information that is in the database about the calling number.
